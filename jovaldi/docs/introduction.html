The jOVAL Definition Interpreter

jOVAL is an implementation of OVAL in the Java programming language. jovaldi is an OVAL definition interpreter that is implemented within the jOVAL library, that is intended to function as a drop-in replacement for MITRE's public-domain reference implementation, ovaldi.

The primary advantage of jovaldi over ovaldi is that jovaldi features a remote-scanning plug-in, and therefore, it need not be installed on the hosts that it scans. For more information, please see the jovaldi product page
